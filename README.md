# Система проверки плагиата в учебно-методических работах сотрудников ИРНИТУ
Система предназначена для загрузки и проверки на наличие заимстований в работах НПР ИРНИТУ для поддержки принятия решений Учебно-методическим советом ИРНИТУ при назначении стимулирующих выплат в рамках эффективного контракта. 
Проверка осуществляется на базе заранее загруженнного и обработанного корпуса текстов путем вычисления эмбеддингов моделью BERT. Единица проверяемого текста - предложение более 5 слов, очищенное от стоп-слов рускоязычного словаря (https://github.com/xiamx/node-nltk-stopwords/blob/master/data/stopwords/russian). Для хранения и работы с эмбеддингами использована вектораня БД Qdrant.

## Использование
Для использования системы перейдите по адресу http://:8000
Далее загрузите файлы, на базе которых будет выполянться проверка работы (корпус текстов) и выполните расчет эмбеддингов для них(обратите внимение, что это может занять достаточно длительное время, не прерывайте этот процесс). 
Затем перейдите на вкладку загрузки работы и загрузите проверяемую работу и также выполните расчет эмбеддингов и ознакомьтесь с отчетом о поиске заимствований.

## Разработка

### Требования
Для установки и запуска проекта, необходим [NodeJS](https://nodejs.org/) v8+.

### Установка зависимостей
Для установки зависимостей, выполните команду:
```sh
$ npm i
```

### Запуск Development сервера
Чтобы запустить сервер для разработки, выполните команду:
```sh
npm start
```

### Создание билда
Чтобы выполнить production сборку, выполните команду: 
```sh
npm run build
```
